# EDUFLOW — Interactive Learning Management System

EDUFLOW is a professional, full-stack Learning Management System (LMS) designed for modern educational environments. It features a robust role-based access control (RBAC) system, a dynamic assessment engine, and comprehensive course management capabilities.

## 🚀 Key Features

- **Multi-Role Portal**: Tailored experiences for Admins, Faculty, and Students.
- **Interactive Course Hub**: Manage materials, take MCQ quizzes, and submit assignments in a Teams-inspired interface.
- **Automated Grading**: Instant evaluation for MCQ assessments and integrated Grade Point (CGPA) computation.
- **Attendance Registry**: Automated eligibility tracking based on student attendance records.
- **File Management**: Secure upload and download systems for course materials and assignment submissions.
- **Real-time Analytics**: Faculty and Admins can monitor student performance and attendance at a glance.

## 🛠️ Technology Stack

| Layer | Technology |

## ⚙️ Development Setup

### 1. Backend Configuration
1. Navigate to the `backend` directory.
2. Initialize a virtual environment and install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt  # Ensure you have requirements.txt or manually install fastapi, uvicorn, mysql-connector-python, PyJWT, python-dotenv
   ```
3. Copy `.env.example` to `.env` and provide your TiDB/MySQL credentials:
   ```bash
   cp .env.example .env
   ```
4. Run the seed script to initialize the database:
   ```bash
   python seed.py
   ```
5. Start the API server:
   ```bash
   uvicorn main:app --reload
   ```

### 2. Frontend Configuration
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🏛️ Database Schema
The system utilizes a relational schema optimized for high performance on TiDB:
- **Users**: Admin, Faculty, and Student accounts.
- **Courses**: Academic modules with assigned faculty.
- **Enrollments**: Linking students to courses with attendance and eligibility tracking.
- **Quizzes & Scores**: Automated MCQ system with JSON-based question storage.
- **Assignments & Submissions**: File-based submission system with due-date logic.
- **Materials**: Centralized course resource repository.
